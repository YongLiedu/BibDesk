BibDesk

Scripts:

This is a list of scripts (mostly Applescripts) that can extend BibDesk's functionality. These scripts are written by BibDesk users and this list keeps track of those posted to the bibdesk-users mailing list.

To add an Applescript to the menu put it in ~/Library/Application Support/BibDesk/Scripts/ (for a single user only, where ~ is the home directory of this user) or in /Library/Application Support/BibDesk/Scripts/ (available for all the users on the machine; might require administrator password). You may have to create those directories if they do not exist. To run an Applescript, select it in the menu.

  • Christiaan Hofman's applescripts [website] [ReadMe]. These scripts include:
    • Capitalize Titles
    • Capitalize Titles
    • Downcase Titles
    • Download from ArXiv
    • Download BibTeX
    • Download BibTeX and PDF
    • Download PDF
    • Set Spires Cite Keys
    • Add Protect Strings
    • Protect Titles
    And an additional applescript to gather references stored into a doc file in a structured manner and put them into a BibDesk document: [download] [ReadMe]
  • Alex Montgomery's field applescripts [website] [ReadMe]. They include:
    • Field Capitalize
    • Field Copy/Swap
    • Field Copy or Swap
    • Field Prefix or Suffix Add
    • Field Put 1 Before or After 2
    • Field Remove Chars from Start or End
    • Field Search&Replace
    • Field Set
  • Simon Spiegel's applescripts [website] [ReadMe]. They include:
    • Addwepage
    • Nebis
    • Amazonsearch
    • RLGSearch
    An addition MSWord (and Pages?) integration bash script [download] [ReadMe].
  • Michael Hohensee's applescripts [website] [ReadMe]. They include:
    • ArXiv reader
    • Safari script to create automatically bib entries from an IOP, OSA, AIP, or APS journal article
